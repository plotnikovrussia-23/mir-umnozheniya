# Мир умножения

Небольшая обучающая игра на `React + Vite` для тренировки таблицы умножения.

## Локальный запуск

```bash
npm install
npm run dev
```

## Продакшен-сборка

```bash
npm run build
```

Собранные статические файлы появляются в каталоге `dist/`.

## Публикация на Replit

Для `Static Deployment` в Replit:

- `Build command`: `npm run build`
- `Public directory`: `dist`

Файл `.replit` уже добавлен в проект с базовой конфигурацией для импорта.
